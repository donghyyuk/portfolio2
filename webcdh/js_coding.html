자바스크립트 코드 작성이 가능한 위치

1. HTML 태그의 이벤트 리스너 속성에 작성
2. 태그에 작성
3. 별도 파일에 작성
4. URL 부분에 작성


자바스크립트 작성 예

1. <img src="apple.png" alt="img" onclick="this.src='banana.png'">
2. <script> document.write("<h3>유수봉 만세</h3>"); </script>
3. <script src="javascriptfile.js"> </script>
4. <a href= "javascript:alert('클릭하셨어요?')" > 클릭해보세요</a>
